Gender

病人性別。數值個數(0…1)，可不提供，或給定以下數值:male | female | other | unknown

議題：

格式如下:

    "gender": "male",
    "birthDate": "1974-12-25",
    "_birthDate": {
      "extension": [
        {
          "url": "http://hl7.org/fhir/StructureDefinition/patient-birthTime",
          "valueDateTime": "1974-12-25T14:35:45-05:00"
        }
      ]
    },
